# Demo Video Script

## Overview

Record a 2-3 minute demo video showing the complete user flow of the WorldID Reward Distribution System. The video should be recorded on a real iOS or Android device.

## Equipment Needed

- iOS or Android device (real device, not simulator)
- Screen recording capability
- Good lighting
- Clear audio (optional, can be silent with text overlays)

## Video Structure

### Introduction (0:00 - 0:15)

**Visual**: Show the device, then open the mini-app

**Narration/Text Overlay**:
```
"WorldID Reward Distribution System
Preventing multi-account abuse with proof of personhood"
```

**Action**: 
- Show QR code scanning (if applicable)
- Or navigate to the app URL

---

### Step 1: Browse Events (0:15 - 0:30)

**Visual**: Show the events list page

**Narration/Text Overlay**:
```
"Browse available events with rewards"
```

**Action**:
- Scroll through events list
- Show event details (name, description, reward count)
- Highlight that events have rewards available

**Key Points to Show**:
- Clean UI
- Event information clearly displayed
- Multiple events available

---

### Step 2: Connect Wallet (0:30 - 0:45)

**Visual**: Show wallet connection flow

**Narration/Text Overlay**:
```
"Connect your wallet to participate"
```

**Action**:
- Click "Connect Wallet" button
- Show MetaMask/wallet connection prompt
- Complete wallet connection
- Show connected wallet address

**Key Points to Show**:
- Smooth wallet connection
- Wallet address displayed
- User is now authenticated

---

### Step 3: Join Event (0:45 - 1:15)

**Visual**: Show event detail page and join flow

**Narration/Text Overlay**:
```
"Join an event with WorldID verification
This links your WorldID to your wallet"
```

**Action**:
1. Navigate to an event detail page
2. Click "Join Event"
3. Show WorldID verification widget
4. Complete WorldID verification (can show the flow)
5. Show success message: "Successfully joined event"

**Key Points to Show**:
- WorldID verification is required
- Privacy notice (if shown)
- Verification completes successfully
- User is now registered for the event

---

### Step 4: Claim Rewards (1:15 - 1:45)

**Visual**: Show reward claiming flow

**Narration/Text Overlay**:
```
"Claim rewards with WorldID verification
One person = one claim per event"
```

**Action**:
1. Scroll to "Claim Rewards" section
2. Click "Verify with WorldID" button
3. Show WorldID verification (can be quick)
4. Show "WorldID Verified" confirmation
5. Click "Claim Rewards" button
6. Show processing/loading state
7. Show success message with transaction hash

**Key Points to Show**:
- Fresh WorldID proof required for each claim
- Verification completes
- Claim is processed
- Transaction hash shown (if applicable)

---

### Step 5: Duplicate Prevention Demo (1:45 - 2:15)

**Visual**: Show attempt to claim again

**Narration/Text Overlay**:
```
"Try to claim again - prevented by WorldID
This is what makes the system fair"
```

**Action**:
1. Try to claim rewards again
2. Show WorldID verification (if required)
3. Show error message: "Rewards for this event have already been claimed"
4. Highlight that the system prevented duplicate claim

**Key Points to Show**:
- System prevents duplicate claims
- Clear error message
- Fair distribution guaranteed

---

### Step 6: Privacy Indicators (2:15 - 2:30)

**Visual**: Show privacy notice or privacy information

**Narration/Text Overlay**:
```
"Privacy protected:
- No identity stored
- Zero-knowledge proofs
- Data minimization"
```

**Action**:
- Show privacy notice (if in UI)
- Or highlight privacy features
- Show what data is/isn't collected

**Key Points to Show**:
- Privacy-by-design
- No personal information collected
- User privacy protected

---

### Conclusion (2:30 - 2:45)

**Visual**: Show summary or final screen

**Narration/Text Overlay**:
```
"WorldID Reward Distribution System
Fair, private, sybil-resistant rewards"
```

**Action**:
- Show final screen
- Highlight key features
- Show app name/logo

## Recording Tips

### Before Recording

1. **Prepare the Environment**:
   - Clear device notifications
   - Close unnecessary apps
   - Ensure good internet connection
   - Test screen recording works

2. **Prepare Test Data**:
   - Create a test event
   - Have test rewards ready
   - Ensure WorldID verification works
   - Test the complete flow

3. **Plan the Flow**:
   - Write down key points to show
   - Practice the flow once
   - Time each section

### During Recording

1. **Keep it Smooth**:
   - Don't rush, but don't be too slow
   - Smooth scrolling and navigation
   - Clear taps/clicks

2. **Show Key Features**:
   - WorldID verification
   - Wallet connection
   - Claim process
   - Duplicate prevention

3. **Highlight Important Points**:
   - Pause briefly on important screens
   - Show error messages clearly
   - Highlight privacy indicators

### After Recording

1. **Edit if Needed**:
   - Trim unnecessary parts
   - Add text overlays if needed
   - Ensure audio is clear (if included)

2. **Export**:
   - Export as MP4 or MOV
   - Ensure good quality
   - Keep file size reasonable (< 100MB)
